# better-strip-color

Strip ansi colors, attributes from a string.

## Install

Install with [npm](https://www.npmjs.com/) or [pnpm]()

```sh
$ npm i better-strip-color
```

## Usage

```js
const strip = require("better-strip-color");
const strippedString = strip(ANSIString);
```
